# Brandingbestanden

Plaats de PNG-brandingbestanden voor deze custom integratie in deze map.

Aanbevolen bestandsnamen:

```text
icon.png
icon@2x.png
logo.png
logo@2x.png
```

Eventueel kunnen ook donkere varianten worden toegevoegd:

```text
icon_dark.png
icon_dark@2x.png
logo_dark.png
logo_dark@2x.png
```

Richtlijnen:

- gebruik PNG-bestanden met transparante achtergrond;
- gebruik voor `icon` een vierkante afbeelding;
- gebruik voor `logo` een bredere versie wanneer beschikbaar;
- houd bestandsnamen exact gelijk aan bovenstaande namen;
- commit de bestanden rechtstreeks in deze map.

Git bewaart geen lege mappen. Dit README-bestand zorgt ervoor dat de map alvast in de repository bestaat. Zodra de afbeeldingen zijn toegevoegd, kan dit bestand blijven staan of worden verwijderd.
