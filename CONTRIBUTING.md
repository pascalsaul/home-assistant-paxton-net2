# Bijdragen

Issues en pull requests zijn welkom nadat het project is gepubliceerd.

Vermeld bij een probleem minimaal:

- Home Assistant-versie;
- Paxton Net2-versie;
- integratieversie;
- relevante Home Assistant-logregels;
- betrokken endpoint en HTTP-status;
- of het probleem een normale controller of PaxLock betreft.

Verwijder altijd wachtwoorden, bearer tokens, persoonsnamen, kaartnummers en andere toegangsgegevens voordat je logbestanden deelt.
